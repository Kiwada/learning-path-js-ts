# Módulos JS ES6 e POO

Este diretório contém projetos de estudo focados em JavaScript Moderno (ES6+), Módulos e Programação Orientada a Objetos.

## Projetos

### 1. mv1 (MVC Project)
Projeto estruturado no padrão MVC (Model-View-Controller) utilizando Node.js, Express e EJS como template engine.
- **Foco**: Estrutura de backend, rotas, controllers e views dinâmicas.

### 2. passwordgen-wbp (Gerador de Senhas)
Aplicação frontend para geração de senhas seguras.
- **Tecnologias**: Webpack, Babel, CSS e JS Modules.
- **Foco**: Configuração de ambiente de desenvolvimento moderno e manipulação de DOM.

### 3. tpircsava 1 (Validação de CPF)
Script simples para validação de CPF.
- **Foco**: Lógica de algoritmos e manipulação de strings/arrays.

### 4. validar-cpf (Validação de CPF com Classe)
Refatoração da validação de CPF utilizando Classes (POO).
- **Foco**: Programação Orientada a Objetos, Classes e Métodos estáticos/instância.
