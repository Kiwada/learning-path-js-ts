# Api-Express-Mongo

This project is an Express.js application integrated with MongoDB.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Navigate to the project directory:
   ```bash
   cd Api-Express-Mongo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:

```bash
npm run dev
```

## Project Structure

- `server.js`: Entry point of the application.
- `package.json`: Project metadata and scripts.
